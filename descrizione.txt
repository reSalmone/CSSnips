📁 CSSNIPS-MAIN (1)/CSSNIPS-MAIN
Contiene i file principali dell’applicazione web.

    about_us.php – Pagina informativa "Chi siamo".
    account.php – Visualizzazione e gestione del profilo utente.
    account_settings.php – Impostazioni account dell’utente.
    activity.php – Mostra le attività recenti dell’utente.
    challenges.php – Elenco delle sfide disponibili.
    challenge_selected.php – Dettagli di una sfida selezionata.
    creator.php – Interfaccia per creare contenuti/snippet.
    explorer.php – Esplora contenuti e snippet.
    footer-code.php – Codice riutilizzabile per il footer.
    index.php – Home page dell’applicazione.
    login-signup-code.php – Codice condiviso tra login e registrazione.
    navbar-code.php – Codice per la barra di navigazione.
    snippet.php – Visualizzazione e gestione di uno snippet.

📁 ASSETS/NOVEOSANS-BOOK
Contiene stili collegati ai font.

    style.css – Foglio di stile per il font NoveoSans.

📁 CSS
Contiene tutti i file CSS che definiscono l’aspetto grafico delle varie sezioni.

    about_us.css – Stile della pagina "Chi siamo".
    account.css, account_settings.css – Stili del profilo utente e impostazioni.
    activity.css – Stile per la pagina delle attività.
    challenges.css, challenge_selected.css – Aspetto grafico delle sfide.
    checkbox.css – Stili personalizzati per le checkbox.
    creator.css – Stile per la sezione di creazione.
    data.css – Stile generico, probabilmente per dati e tabelle.
    explorer.css – Layout della sezione esplora.
    footer.css, navbar.css – Aspetto di footer e navbar.
    index.css – Stile della homepage.
    login-signup.css – CSS per login e registrazione.
    snippet-loader.css – Stili per elementi di caricamento.
    snippet.css – Stile visivo degli snippet.

📁 HANDLERS
Contiene file PHP che gestiscono operazioni lato server (autenticazione, like, upload...).

    account_follow_action.php – Gestione follow/unfollow.
    checkname.php, checkusername.php – Verifica nome e username.
    delete-*.php – File per eliminare commenti, draft, like, ecc.
    like.php, like-comment.php, like_snip.php – Gestione dei like.
    load_snippets.php – Carica snippet dal database.
    login.php, logout.php, signup.php – Login, logout e registrazione.
    save.php, update_*.php, upload.php* – Salvataggio e aggiornamenti.

📁 SCRIPTS
Contiene file JavaScript che gestiscono le funzionalità interattive dell’app.

    account.js, account_settings.js – Interazione con il profilo utente.
    challenges.js, challenge_selected.js – Logica delle sfide.
    creator.js, explorer.js – Script per creare ed esplorare contenuti.
    index.js – Script generali della home.
    login.js, signup.js – Script per login e registrazione.
    sidebar.js – Gestione della barra laterale.
    snippet.js – Gestione snippet e interazioni.
    title-wave.js, title-wave2.js – Effetti animati nei titoli.
